Instalación de Proxmox VE como base de laboratorio

Proxmox VE fue la base elegida para desplegar una infraestructura virtualizada donde separar servicios, redes y funciones de seguridad.

Proxmox VE Virtualización Linux Infraestructura IT

Objetivo

El objetivo era disponer de una plataforma de virtualización flexible para crear máquinas virtuales y contenedores, separando servicios como firewall, servidor web, base de datos y automatización.

Por qué Proxmox VE

Proxmox VE permite gestionar máquinas virtuales y contenedores desde una interfaz web, facilitando la administración de recursos, redes virtuales, almacenamiento y servicios desplegados.

  • Permite trabajar con máquinas virtuales y contenedores.
  • Facilita la creación de laboratorios técnicos.
  • Ofrece una interfaz de administración clara.
  • Encaja bien con proyectos de sistemas, redes y seguridad.

Planteamiento del entorno

La instalación se planteó sobre un equipo físico usado como servidor de laboratorio. Desde Proxmox se desplegaron los distintos elementos del proyecto: firewall, servicios web, base de datos y contenedores auxiliares. 

Servidor físico
   └── Proxmox VE
        ├── Firewall IPFire
        ├── Servidor web
        ├── Base de datos
        └── Contenedor para automatización

Competencias demostradas

  • Instalación de un hipervisor de virtualización.
  • Planificación de recursos hardware.
  • Organización de servicios mediante máquinas y contenedores.
  • Base para segmentación de red y despliegue seguro.

Conclusión

Proxmox VE permitió construir un entorno controlado, flexible y escalable para desplegar el resto de la infraestructura del proyecto.
